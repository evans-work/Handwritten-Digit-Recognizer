# Handwritten Digits Recognizer

## A python script that uses the **mnist** dataset to create a model that can recognize digits in 28X28 images.

## Running the script
1. Dependancies
- numpy
- tensorflow
- matplotlib
- python-opencv

2. Command
- py Main.py or python Main.py
